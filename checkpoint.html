Summary

We currently build the entire scene on Shadertoy. Overall, we aim to build a sweeping lighthouse beam that lights an animated ocean and a buoyant boat. The scene is built with signed-distance functions and a ray-marcher, avoiding mesh overhead. Five octaves of hash-based value noise drive wave heights; Fresnel shading blends water color with sky reflections. Boat buoyancy samples corner heights, while a cross-product frame tilts the hull to match wave slopes. The beam casts soft shadows via the same distance field.

Preliminary Results

We developed initial water simulation, buoyancy and basic boat simulation, and sky and cloud simulation on Shadertoy.
