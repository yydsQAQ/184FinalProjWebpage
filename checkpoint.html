Summary

Our project combines ray-based rendering with physics simulation to create dynamic visual effects. Using Signed Distance Functions and Raymarching instead of traditional vertex geometry, we aim to develop a realistic water system that interacts with a simulated boat. Our current implementation calculates floating height by sampling water at object corners, creates boat orientation changes through vector-based calculations to simulate tilting on waves, and generates animated water movement using layered noise functions. Eventually, we want to build a scene that features a lighthouse (not yet created) with dynamic rotating light beams illuminating the water surface, atmospheric effects including gradient skies, and drifting cloud layers.

Preliminary Results

We developed initial water simulation, buoyancy and basic boat simulation, and sky and cloud simulation on shadertoy. Further details are in the presentation slides.
